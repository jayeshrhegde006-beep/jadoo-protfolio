# 👨‍💻 Jayesh Hegde Portfolio

Welcome to the **Jayesh Hegde Portfolio** repository.  
This project is a modern personal portfolio website designed to showcase my **skills, projects, technical expertise, and development experience** across multiple technologies.

The portfolio highlights my work in **web development, AI-based applications, game development, and 3D graphics**, providing visitors with an interactive platform to explore my projects and technical capabilities.

---

# 🚀 Features

- Modern responsive portfolio website
- Interactive UI with smooth animations
- 3D graphics and motion effects
- AI-powered project integrations
- Project showcase with detailed tech stacks
- Contact and communication integration

---

# 🖥️ Frontend Technologies

- **Frameworks:** React.js (v19), Next.js (v16)  
- **Build Tools:** Vite, TypeScript  
- **Styling & UI:** Tailwind CSS (v4), Shadcn/UI, Glassmorphism, CSS3  
- **Mobile Development:** Flutter  

---

# ⚙️ Backend Technologies

- **Runtime Environment:** Node.js  
- **APIs:** Next.js API Routes, WhatsApp API  
- **Languages:** C++, C# (Unity scripting)  
- **Logic & Automation:** Python (used for desktop and database integration)

---

# 🗄️ Database Technologies

- **SQL Databases:** MySQL, SQLite  
- **NoSQL Databases:** MongoDB  
- **File-Based Storage:** JSON (used for structured data like `questions.json`)
